# PayFlow - Digital Wallet Application

![PayFlow Logo](public/logo.png)

PayFlow is a modern, secure digital wallet application designed to simplify financial transactions for users, agents, and administrators. Built with React, TypeScript, and Vite, it offers instant money transfers, comprehensive transaction tracking, and bank-grade security. Whether you're sending money to friends, depositing funds via agents, or managing a network of users, PayFlow provides a seamless and intuitive experience.
